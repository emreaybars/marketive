Çarkıfelek Widget - Embed Örneği

⚠️ Önemli: Widget'ı kullanmak için aşağıdaki değerleri kendi Supabase bilgilerinizle değiştirmeniz gerekiyor.

1. HTML Container (Zorunlu)

Widget'ın görüntüleneceği container elementi:

<!-- Widget Container -->
<div id="carkifelek-widget"
     data-supabase-url="YOUR_SUPABASE_URL"
     data-supabase-key="YOUR_SUPABASE_ANON_KEY"
     data-token="YOUR_SHOP_TOKEN"
     data-shop-id="YOUR_SHOP_ID">
</div>

2. Widget Script

Widget script'ini container'dan SONRA ekleyin:

<script src="https://your-domain.com/widget.js"></script>

3. Tam Örnek

<!DOCTYPE html>
<html>
<head>
  <title>Çarkıfelek Widget</title>
</head>
<body>

  <!-- Widget Container -->
  <div id="carkifelek-widget"
       data-supabase-url="https://xxx.supabase.co"
       data-supabase-key="eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
       data-token="eyJzaWQiOiJzaG9wLTEiLCJ1aWQiOiJ1dWlkLTEyMyIsInRzIjoxNjM5...="
       data-shop-id="shop-123">
  </div>

  <!-- Widget Script -->
  <script src="https://your-domain.com/widget.js"></script>

</body>
</html>

4. Parametre Açıklamaları

Parametre Açıklama
id="carkifelek-widget" Widget container ID (zorunlu)
data-supabase-url Supabase proje URL
data-supabase-key Supabase anon/public key
data-token Shop token (backend'den alın)
data-shop-id Shop ID

5. Token Oluşturma

Shop token oluşturmak için backend API'yi kullanın:

// Backend'de token oluşturma
const { generateShopToken } = require('./server/widget-api.js');
const token = generateShopToken('shop-123', 'uuid-here');

// Embed kodu
const embedCode = `
<div id="carkifelek-widget"
     data-supabase-url="${process.env.SUPABASE_URL}"
     data-supabase-key="${process.env.SUPABASE_ANON_KEY}"
     data-token="${token}"
     data-shop-id="shop-123">
</div>
<script src="/widget.js"></script>
`;
